1. You are given a string S, and your task is to return an array B(having a size of 2), where B[0] contains the count of character A (uppercase) in string S and B[1] contains the count of character D (uppercase) in string S.

Source Code:

        const countCharacters = (S) => 
        {
            let count=0;
            let sum=0;
            let B=[];
            for(let i=0;i<S.length;i++){
              if(S[i]==="A"){
                count++;
              }
              else if(S[i]==="D"){
                sum++;
              }
            }
            B.push(count);
            B.push(sum);
            return B;
        };

2. Tina is given a string S which contains the first letter of all the student names in her class. She got a curiosity to check how many people have their names starting from the same alphabet. So given a string S , she decided to write a code that finds out the count of characters that occur more than once in the string.

Source Code:

      var Count_Occ = (s) => 
      {
        let newstr="";
        s=s.toLowerCase().split("").sort().join("");
        for(let i=0;i<s.length-1;i++){
          let count=0;
          for(let j=i+1;j<s.length;j++){
            if(s[i]===s[j]){
              count++;
            }
          }
          if(count>0 && ! newstr.includes(s[i])){
            newstr+=s[i];
            newstr+=count+1;
          }
        }
        return newstr;
      };
